# Brew & Bloom 🌸

An elegant floral showcase and ordering website built for a home-based flower business. Brew & Bloom provides customers with a seamless way to explore handcrafted bouquets, view floral collections, and connect directly for orders.

## Live Website

Production URL: https://brew-and-bloom-sigma.vercel.app

## Overview

Brew & Bloom is a modern florist website designed to:

* Showcase floral arrangements through a curated gallery
* Present bouquet collections in an organized format
* Build trust through testimonials
* Explain the ordering process clearly
* Enable direct customer contact and inquiries
* Deliver a premium, mobile-friendly browsing experience

The website is optimized for both desktop and mobile devices and emphasizes visual presentation, simplicity, and accessibility.

## Local Development

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Deployment

The project is deployed on Vercel with automatic deployments enabled.

Every push to the `main` branch triggers:

1. GitHub update
2. Automatic Vercel build
3. Production deployment

```bash
git add .
git commit -m "Update website"
git push
```
